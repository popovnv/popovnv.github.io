ALTER LARGE OBJECT

Название

ALTER LARGE OBJECT -- изменить определение большого объекта

Синтаксис

ALTER LARGE OBJECT oid_большого_объекта OWNER TO новый_владелец

Описание

ALTER LARGE OBJECT изменяет определение большого объекта. Единственное, что может сделать эта команда — назначить нового владельца. Выполнить ALTER LARGE OBJECT может только суперпользователь или владелец большого объекта.

Параметры

oid_большого_объекта

OID изменяемого большого объекта

новый_владелец

Новый владелец большого объекта

Совместимость

Оператор ALTER LARGE OBJECT отсутствует в стандарте SQL.
