60.4. Пример

Следующая последовательность команд создаст таблицу test_table с OID 420, имеющую две колонки cola и colb типа int4 и text, соответственно, и вставит две строки в эту таблицу: 

create test_table 420 (cola = int4, colb = text)
open test_table
insert OID=421 ( 1 "value1" )
insert OID=422 ( 2 _null_ )
close test_table
