34.17. constraint_column_usage

Представление constraint_column_usage описывает все колонки в текущей базе данных, связанные с некоторым ограничением. В нём показываются только колонки таблиц, принадлежащих текущей активной роли. Для ограничений-проверок это представление содержит колонки, задействованные в выражении проверки. Для ограничений внешнего ключа оно содержит колонки, на которые ссылается внешний ключ, а для ограничений уникальности или первичного ключа — ограничиваемые колонки.
