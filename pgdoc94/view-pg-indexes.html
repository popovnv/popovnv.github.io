48.59. pg_indexes

Представление pg_indexes даёт доступ к полезной информации обо всех индексах в базе данных.

Таблица 48-60. Колонки pg_indexes

ИмяТипСсылкиОписание
schemaname name pg_namespace.nspname Имя схемы, содержащей таблицу и индекс
tablename name pg_class.relname Имя таблицы, для которой создан индекс
indexname name pg_class.relname Имя индекса
tablespace name pg_tablespace.spcname Имя табличного пространства, содержащего индекс (NULL, если это пространство по умолчанию)
indexdef text  Определение индекса (реконструированная команда CREATE INDEX)
