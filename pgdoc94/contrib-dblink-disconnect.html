dblink_disconnect

Название

dblink_disconnect -- закрывает постоянное подключение к удалённой базе данных

Синтаксис

dblink_disconnect() returns text
dblink_disconnect(text connname) returns text

Описание

dblink_disconnect() закрывает подключение, ранее открытое функцией dblink_connect(). Форма без аргументов закрывает безымянное подключение.

Аргументы

conname

Имя закрываемого именованного подключения.

Возвращаемое значение

Возвращает состояние (это всегда строка OK, так как в случае любой ошибки функция прерывается, выдавая исключение).

Примеры

SELECT dblink_disconnect();
 dblink_disconnect
-------------------
 OK
(1 row)

SELECT dblink_disconnect('myconn');
 dblink_disconnect
-------------------
 OK
(1 row)
