15.3. Получение исходного кода

Исходные коды PostgreSQL 9.4.1 можно загрузить из соответствующего раздела нашего сайта: http://www.postgresql.org/download/. Вам следует выбрать файл postgresql-9.4.1.tar.gz или postgresql-9.4.1.tar.bz2. Получив этот файл, распакуйте его: 

gunzip postgresql-9.4.1.tar.gz
tar xf postgresql-9.4.1.tar

(Если вы выбрали файл .bz2, используйте bunzip2 вместо gunzip.) При этом в текущем каталоге будет создан подкаталог postgresql-9.4.1 с исходными кодами PostgreSQL. Перейдите в этот подкаталог для продолжения процедуры установки.

Вы также можете получить исходный код непосредственно из репозитория системы управления версиями (см. Приложение H).
