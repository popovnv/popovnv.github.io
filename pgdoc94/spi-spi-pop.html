SPI_pop

Название

SPI_pop -- опустить стек SPI для выхода из рекурсивного использования SPI

Синтаксис

void SPI_pop(void)

Описание

SPI_pop восстанавливает предыдущее окружение из стека вызовов SPI. См. SPI_push.
