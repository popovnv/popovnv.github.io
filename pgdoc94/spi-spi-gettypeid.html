SPI_gettypeid

Название

SPI_gettypeid -- получить OID типа данных указанной колонки

Синтаксис

Oid SPI_gettypeid(TupleDesc rowdesc, int colnumber)

Описание

SPI_gettypeid возвращает OID типа данных указанной колонки.

Аргументы

TupleDesc rowdesc

описание строк

int colnumber

номер колонки (начиная с 1)

Возвращаемое значение

OID типа данных указанной колонки, либо InvalidOid в случае ошибки. При ошибке в SPI_result устанавливается SPI_ERROR_NOATTRIBUTE.
