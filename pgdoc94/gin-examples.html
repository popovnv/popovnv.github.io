58.7. Примеры

Дистрибутив исходного кода PostgreSQL содержит классы операторов GIN для tsvector и для одномерных массивов всех внутренних типов. Поиск по префиксу в tsvector реализован с применением возможности проверки частичного соответствия GIN. Классы операторов GIN также содержатся в следующих дополнительных модулях (contrib):

btree_gin

Функциональность B-дерева для различных типов данных

hstore

Модуль для хранения пар (ключ, значение)

intarray

Расширенная поддержка int[]

pg_trgm

Схожесть текста на основе статистики триграмм
