34.55. udt_privileges

Представление udt_privileges описывает права USAGE, назначенные текущей активной роли или текущей активной ролью для пользовательских типов. Оно содержит отдельную строку для каждой комбинации типа, праводателя и правообладателя. Это представление показывает только составные типы (почему, говорится в Разделе 34.57); права, затрагивающие домены, описаны в Разделе 34.56.

Таблица 34-53. Колонки udt_privileges

ИмяТип данныхОписание
grantor sql_identifier Имя роли, давшей право (праводатель)
grantee sql_identifier Имя роли, которой было дано право (правообладатель)
udt_catalog sql_identifier Имя базы данных, содержащей тип (всегда текущая база)
udt_schema sql_identifier Имя схемы, содержащей тип
